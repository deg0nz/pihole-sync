# pihole-sync

A (currently) quick and dirty utility to sync your PiHole v6 configuration to multiple PiHole instances.
The sync goes one-to-many. One main instance is specified and it's configuration is distributed to all other (secondary) instances.

## Features

- Syncs everything contained in PiHole's Teleporter backups
- Acquire app passwords for PiHole API
- Modify and add PiHole instances via CLI

## Installation

- Currently only available via `git clone` and `cargo build`
